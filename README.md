# Prostate MRI Dataset Benchmark

This repository provides the benchmark and validation code used in our study for three prostate MRI datasets with healthy controls, periprostatic adipose tissue (PPAT) segmentation, and longitudinal follow-up.

We include implementations of representative methods for:
- Semi-supervised segmentation (Dataset 1)
- Supervised segmentation (Dataset 2)
- Unsupervised deformable registration (Dataset 3)

These implementations are adapted from publicly available open-source projects and are provided to facilitate reproducibility and further research.


Please refer to the original publications for these methods.
